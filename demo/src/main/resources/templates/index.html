프로젝트 개요

  • 고객 정보를 효율적으로 관리하기 위한 웹 기반 고객 관리 시스템 구축
  • Spring Boot와 MySQL을 활용한 백엔드 API 서버 개발
  • JavaScript, jQuery, Thymeleaf를 이용한 웹 애플리케이션 UI 구현
  • AWS Elastic Beanstalk을 통한 클라우드 배포 및 운영

프로젝트 내용

  • 고객 정보 등록 / 조회 / 수정 / 삭제 기능 제공
  • 검색 기능 (이름, 성별, 전화번호, 이메일 등)

기대 효과

  • 고객 정보를 체계적으로 관리하여 검색 및 접근성 개선
  • API 기반 설계를 통해 유연한 확장 가능
